FUENTE DE LOS DATOS Y PERMISOS

Esta App utiliza información extraída directamente de la web de Red Eléctrica de España (REE) y otros cálculos de elaboración propia a partir de esos datos.

Para la descarga de esos datos se utiliza principalmente la API oficial de REE, organismo que en torno a las 20:15h de cada día publica los precios de la electricidad que se aplicarán al día siguiente. Por tanto, esta aplicación depende de la disponibilidad de ese servidor y requiere conexión a internet para recabar la información.

Esta aplicación no tiene ninguna vinculación oficial con REE.

Para la obtención de la etiqueta energética, esta aplicación se conecta a la base de datos EPREL (Registro de la Comisión Europea de productos para el etiquetado energético).
